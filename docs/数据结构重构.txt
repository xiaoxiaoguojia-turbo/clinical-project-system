现在有八大类项目类型，字段为：
------------------------------------通用字段------------------------------------
{
  _id，主键ObjectId
  department，临转院部门（转移转化与投资一部、转移转化与投资二部、转移转化与投资三部），必填，默认值为转移转化与投资一部
  name，项目名称，必填
  projectType，项目分类型（包括：院内制剂、AI医疗及系统研究、检测诊断、细胞治疗、药物、医疗器械、医用材料、其他），必填，默认值为院内制剂
  source，项目来源，必填
  重要程度（非常重要、重要、一般、不重要），必填，默认值非常重要
  status，项目进展阶段（早期，临床前，临床阶段，上市产品），必填，默认值早期
}
------------------------------------通用字段------------------------------------
+
------------------------------------特有字段------------------------------------
"院内制剂":
{
  composition，组方，必填
  function，功能，必填
  specification，规格，非必填
  duration，年限，非必填
  dosage，用量，非必填
  recordNumber，备案号，不需要设置为为unique，非必填
  remarks，备注，非必填
}

"除院内制剂外的所有类型(AI医疗及系统研究、检测诊断、细胞治疗、药物、医疗器械、医用材料、其他)":
{
  leader，负责人，必填
  startDate，开始日期，必填
  适应症/科室，非必填
  跟进时间/周，必填
  转化需求（许可转让、代价入股、代持、代持托管、公司化运营、许可转让现金、待定），必填，默认值待定
  院端医生，非必填
  项目结论，非必填
}
------------------------------------特有字段------------------------------------
+
------------------------------------通用字段------------------------------------
{
  patent，专利信息，非必填
  临床数据，非必填
  市场规模，非必填
  竞品状态，非必填
  attachments，附件_id数组
  createTime，创建时间
  updateTime，更新时间
  createdBy，创建人
  aiReport，AI报告
}
